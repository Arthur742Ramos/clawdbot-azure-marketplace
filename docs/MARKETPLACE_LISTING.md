# Azure Marketplace Listing - OpenClawVM

## Offer Name
**OpenClawVM**

## Short Description (256 chars max)
Pre-configured Ubuntu VM with Clawdbot AI agent gateway and coding tools. Supports Claude, GPT, Gemini, GitHub Copilot, and more. Connect to WhatsApp, Telegram, Discord, or use directly via SSH.

## Long Description

### Overview
OpenClawVM is a ready-to-run Ubuntu virtual machine pre-loaded with everything you need to run AI agents in the cloud:

- **Clawdbot Gateway** - AI agent orchestration with multi-channel messaging
- **OpenCode** - Coding agent supporting multiple LLM providers
- **Codex CLI** - OpenAI's coding assistant
- **Claude Code** - Anthropic's coding agent
- **GitHub Copilot CLI** - GitHub's AI pair programmer

### Supported LLM Providers
Choose your preferred AI provider during setup:
- Anthropic (Claude)
- OpenAI (GPT-4, Codex)
- Google (Gemini)
- GitHub Copilot
- OpenRouter
- MiniMax
- Qwen
- And more...

### Messaging Channels
Connect your agent to:
- WhatsApp (personal account linking)
- Telegram (bot or user account)
- Discord (bot integration)
- Slack (app integration)
- Direct CLI/API access

### Getting Started
1. Deploy the VM from Azure Marketplace
2. SSH into your instance: `ssh <username>@<public-ip>`
3. Run the setup wizard: `clawdbot onboard`
4. Choose your LLM provider and configure channels
5. Start chatting with your AI agent!

### Use Cases
- **Personal AI Assistant** - 24/7 AI that knows your context
- **Development Companion** - AI coding agent for your projects
- **Automation Hub** - Schedule tasks, monitor services, manage workflows
- **Multi-Channel Bot** - Single AI across WhatsApp, Telegram, and Discord

### What's Included
- Ubuntu 24.04 LTS (minimal, hardened)
- Node.js 22 LTS
- Clawdbot (latest stable)
- OpenCode, Codex CLI, Claude Code
- GitHub CLI with Copilot extension
- Playwright browsers for web automation
- Systemd service for auto-start

### Requirements
- Azure subscription
- SSH key pair for access
- LLM provider credentials (API key or OAuth)

### Pricing
This VM image is free. You pay only for Azure compute resources:
- Recommended: Standard_B2s ($13/month) or Standard_B2ms ($30/month)
- Minimum: Standard_B1s ($7/month) for light usage
